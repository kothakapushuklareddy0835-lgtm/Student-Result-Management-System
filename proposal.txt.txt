PROJECT DESCRIPTION
=================================================================

The Student Result Management System is a console-based application 
developed in the C programming language. It helps educational institutions 
store, manage, and analyse student academic performance.

The system provides functionality to:
- Add, edit, delete, and search student records
- Calculate percentages and letter grades automatically
- Generate rank lists based on performance
- Compute subject-wise class averages
- Identify at-risk students using rule-based AI

=================================================================
AI FEATURE DETAILS (Rule-Based Expert System)
=================================================================

Type: Rule-Based AI (Heuristic Threshold Detection)

Algorithm:
1. Calculate the class average for each subject
2. Set threshold = class_avg × 0.85 (15% below average)
3. Flag students with marks < threshold
4. Generate personalised improvement suggestions

Why 15%? Represents one standard deviation in educational analytics.
